# 法院诉讼结算管理系统

《法院诉讼结算管理系统》为加强和规范人民法院诉讼费的管理与使用开发的一款软件。 1、预收管理：预收登记、预收查询 2、结算管理：结算登记、结算查询 3、退费管理：退费登记、退费查询 4、统计汇总：统计汇总

# 如何运行本系统?

本系统基于宏达数据库信息管理开发平台开发，下载后，双击文件夹中"dbimp.exe"文件即可完整运行对应的信息管理系统。
